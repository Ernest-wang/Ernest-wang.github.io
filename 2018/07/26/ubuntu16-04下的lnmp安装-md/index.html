PHP

添加软件源

1. apt -y install software-properties-common 
2. add-apt-repository ppa:ondrej/php

安装软件

1. apt install php7.2-fpm php7.2-mysql php7.2-curl php7.2-gd php7.2-mbstring php7.2-xml php7.2-xmlrpc php7.2-zip php7.2-opcache -y

2. sed -i 's/;cgi.fix_pathinfo=1/cgi.fix_pathinfo=0/' /etc/php/7.2/fpm/php.ini

常用命令

systemctl restart php7.2-fpm #重启 
systemctl start php7.2-fpm #启动 
systemctl stop php7.2-fpm #关闭 
systemctl status php7.2-fpm #检查状态
